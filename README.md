# Sistema de Pesquisa Clínica – Constipação

Este projeto é um sistema web para registro e acompanhamento de pacientes em estudo clínico sobre constipação intestinal.

O sistema permite que pesquisadores registrem pacientes, acompanhem evolução clínica e analisem dados para pesquisa.

## Funcionalidades

- Login individual para pesquisadores
- Cadastro completo de pacientes
- Código automático do paciente para anonimização
- Escala de Bristol interativa
- Cálculo automático dos critérios de Roma IV
- Registro de hábitos alimentares, hidratação e atividade física
- Registro de medicamentos e tratamentos
- Avaliação pré e pós procedimento
- Agenda de consultas
- Registro de evolução clínica
- Dashboard com gráficos
- Comparação entre grupos de estudo
- Auditoria de alterações por pesquisador
- Exportação de dados para CSV

## Grupos do estudo

- Grupo 1 – Teste de Fibras
- Grupo 2 – Hidrocolonterapia Isolada
- Grupo 3 – Intervenção combinada

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Supabase (Banco de dados e autenticação)
- GitHub Pages (Hospedagem)
